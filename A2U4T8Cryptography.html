Weakness: Frequency Analysis

In English, 'E' is the most common letter. If a hacker sees one letter appearing frequently in your code, they can guess the shift in seconds!

XOR Binary Sandbox

Rule: $1 \oplus 1 = 0 \mid 0 \oplus 0 = 0 \mid 1 \oplus 0 = 1$
Plaintext Binary (e.g. 'O') DEC: 79
Secret Key (8-bit) DEC: 243
Resulting Ciphertext DEC: 188
Try this: Use the same Key on the Ciphertext. You will get the Plaintext back! This is why XOR is common but must have a long, secret key.
